# 🌐 Desarrollo de Aplicaciones en Internet

> Proyecto académico enfocado en el diseño y desarrollo de aplicaciones web modernas utilizando tecnologías actuales del lado del cliente y servidor.

---

## Descripción

Este repositorio contiene el desarrollo de una aplicación web como parte del curso **Desarrollo de Aplicaciones en Internet**.  
El objetivo es aplicar conceptos fundamentales como:

- Estructura de aplicaciones web  
- Diseño de interfaces (UI/UX)  
- Programación con JavaScript  
- Consumo de APIs  
- Manejo de datos  

---

## Tecnologías Utilizadas

- 🧱 **HTML5** – Estructura de la aplicación  
- 🎨 **CSS3** – Estilos y diseño responsivo  
- ⚙️ **JavaScript** – Lógica e interactividad  
- 🌐 **Bootstrap 5** – Diseño moderno y adaptable  
- 🗄️ **Base de Datos** – (MySQL / MongoDB / según tu proyecto)
